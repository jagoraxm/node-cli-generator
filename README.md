## Generador de proyectos NPM 

Este proyecto servirá para crear proyectos javascript y typescript.

# Requirements
* Node >= 10.15.3 o mayor
* NPM >= 6.0.0

# Install
```npm install -g cli-generator-jag```
# Usage
Ejecutamos la siguiente orden
```npm-cli-creator-jag```
